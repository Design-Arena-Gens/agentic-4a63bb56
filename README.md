# Agentic Studio

Autonomously generated Next.js experience that turns raw ideas into launch-ready narratives with an interactive creative console.

## Quickstart

1. Install dependencies
   ```bash
   npm install
   ```
2. Run the local dev server
   ```bash
   npm run dev
   ```
3. Visit `http://localhost:3000` to explore the studio.

## Tech Stack

- Next.js 14 (App Router)
- React 18 with client-side interaction
- TypeScript with strict settings
- Custom CSS via `styled-jsx` and global styles

## Deployment

The project is production-ready for Vercel:

```bash
npm run build
npm start
```

For Vercel CLI users:
```bash
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-4a63bb56
```

## Project Structure

```
src/
  app/
    layout.tsx       # Global layout + metadata
    page.tsx         # Main experience
    globals.css      # Baseline theming
next.config.mjs      # Next.js configuration
tsconfig.json        # TypeScript configuration
```

## Creative Console

The hero CTA and console encourage fast ideation:
- Pre-filled prompts to jump-start exploration
- Autogenerated responses that adapt to your query
- Feature and timeline sections that clarify the product story

Feel free to remix components, extend the response engine, or plug in real APIs.
